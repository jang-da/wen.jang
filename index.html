데이터 시각화 대시보드

복잡한 데이터를 사용자가 쉽게 이해하고 인사이트를 얻을 수 있도록 설계한 반응형 웹 대시보드입니다. 사용자 리서치를 통해 핵심 지표를 정의하고, D3.js를 활용하여 인터랙티브한 차트를 구현했습니다.

  • 실시간 데이터 연동 및 동적 차트 생성
  • 다양한 필터와 정렬 기능으로 데이터 탐색 지원
  • 모바일 환경에서도 최적화된 반응형 UI/UX
