카제로스 레이드 웹 리디자인

이 프로젝트는 게임 '로스트아크'의 '카제로스 레이드' 업데이트를 주제로 한 인터랙티브 단일 페이지 웹사이트입니다. 사용자가 스크롤하며 콘텐츠를 탐색할 때 동적인 애니메이션과 시각 효과를 통해 몰입감 높은 경험을 제공하는 것을 목표로 제작되었습니다. 최신 웹 기술을 활용하여 사용자의 상호작용에 즉각적으로 반응하고, 다양한 화면 크기에서도 콘텐츠가 아름답게 보일 수 있도록 반응형으로 설계되었습니다.

  • GSAP (GreenSock Animation Platform) & ScrollTrigger
  • Intersection Observer API
  • 순수 JavaScript 기반 슬라이더 및 반응형 처리
